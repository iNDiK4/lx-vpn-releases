## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🚀 **VLESS + XTLS** | Современный протокол с максимальной скоростью |
| 🛡️ **Kill Switch** | Блокировка интернета при обрыве VPN |
| 🔒 **DNS-over-HTTPS** | Защищённые DNS запросы через Cloudflare |
| 🌐 **Split Tunneling** | Выборочный обход блокировок |
| 🌈 **RGB Подсветка** | Анимированная рамка окна |
| 🔄 **Авто-обновления** | Автоматическая проверка новых версий |
| ⚡ **Автозапуск** | Запуск с Windows и автоподключение |

## 🖥️ Системные требования

- Windows 10/11 (64-bit)
- .NET Framework 4.8
- ~50 MB свободного места

## 📥 Установка

1. Скачайте последнюю версию из [Releases](https://github.com/iNDiK4/lx-vpn-releases/releases/latest)
2. Запустите установщик `LXVPN_Setup.exe`
3. Следуйте инструкциям мастера установки
4. Готово! 🎉

## 🔧 Быстрый старт

1. Запустите **LX VPN**
2. Нажмите кнопку **Подключиться**
3. Наслаждайтесь свободным интернетом! 🌍
